# Chapter 3 — Data & Preprocessing Pipeline

## 3.2 Modular Preprocessing Steps

| Step | Function | Notes |
|------|---------|-------|
| Load image | `load_image(path)` | Returns BGR numpy array |
| Grayscale conversion | `to_grayscale(img)` | Handles both BGR and already-grey inputs |
| ROI extraction | `extract_roi(img)` | Variance-based thresholding (block 11×11), morphological closing (25×25 ellipse), bounding-box crop + 5px buffer |
| Contrast enhancement | `enhance_image(img)` | CLAHE (clip=2.0, tile=8×8) |
| Gabor filtering | `apply_gabor_filter(img)` | 4-orientation bank (0°, 45°, 90°, 135°); σ=5.0, λ=10.0, γ=0.5 |
| Resize | `resize_image(img)` | Target: **128×128** |
| Normalise | `normalize_image(img)` | Scale to [0, 1] float32 |

## 3.3 Pipeline Composition

```python
from preprocessing.pipeline import PreprocessingPipeline

pipe = PreprocessingPipeline(img_size=(128, 128))

# Single image (inference / API)
arr = pipe.process(raw_img, augment=False)  # → float32 (128, 128)

# Entire directory (offline training preprocessing)
pipe.process_directory("data/SOCOFing/Real", "data/processed_png/")
```

Output format: grayscale PNG files on disk (lossless, ~128×128 pixels).

## 3.4 Dataset Splitting

Subject-wise split — no subject appears in more than one partition:

| Split | Subjects | Purpose |
|-------|---------|---------|
| Train | 70% | Pair generation for contrastive loss training |
| Val | 15% | Early stopping, LR scheduling |
| Test | 15% | Final metrics: accuracy, FAR, FRR, EER |

Generated by `data/split.py` → saved to `data/metadata/metadata.csv`.

### Metadata CSV schema

| Column | Type | Example |
|--------|------|---------|
| `subject_id` | int | `100` |
| `sex` | str | `M` / `F` |
| `hand` | str | `Left` / `Right` |
| `finger_name` | str | `index` / `little` |
| `alteration_type` | str | `real` / `CR` / `Obl` / `Zcut` |
| `difficulty` | str | `real` / `easy` / `medium` / `hard` |
| `filename` | str | `100__M_Left_index_finger.BMP` |
| `full_path` | str | Absolute path to raw file |
| `split` | str | `train` / `val` / `test` |
