# frontend/

> **Status: Phase 7 — Not yet implemented.**

This directory will contain the NINAuth React/Next.js frontend:

- **Citizen Portal** — Registration and self-service verification
- **Verifier Portal** — Institution-side verification dashboard
- **Admin Dashboard** — System monitoring and audit logs

## Planned Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- ethers.js (MetaMask / wallet integration)
- React Hook Form + Zod

## Quick Start (future)

```bash
cd frontend/
npm install
npm run dev   # http://localhost:3000
```
